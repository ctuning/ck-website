Please check our new beta browser for CK components!


You can detect installed software and register it in the CK as follows: 
 ck pull repo:{Repo UOA - see below}
 ck detect soft:{Soft UOA - see below}
using tags: 
 ck detect soft --tags={some tags from below}
in an unusual path: 
 ck detect soft:{Soft UOA - see below} --search_dirs={path to this software}
or for a different OS target (Android): 
 ck ls os:android* | sort
 ck detect soft:{Soft UOA - see below} --target_os={OS UOA from above}
You can see or use registered virtual CK environments as follows: 
 ck show env
 ck show env --tags={some tags from below}

 ck virtual env:{UID from above}
 ck virtual env --tags={some tags from below}

See 

ck detect soft --help
for more detection options. See related CK packages, CK documentation, "how to contribute" guide, ACM ReQuEST-ASPLOS'18 report and the latest CK paper for further details.
